# Vídeo e Música de Ninar para Crianças no Youtube

## 📒 Descrição
Esse projeto atende a uma necessidade pessoal primeiramente para depois atender ao público, cria uma música com vídeo baseado na história de uma "Gatinha", um pedido especial da minha filha de 2 anos que sempre pede temas na hora de dormir, e que usei ferramentas de IA para criar.

## 🤖 Tecnologias Utilizadas
Notebook LLM, SUNO, Nano Banana, Animaker.

## 🧐 Processo de Criação
Letra gerada com o Notebook LLM a partir de contextos, palavras e rimas defindas por mim,
Música gerada na IA generativa SUNO,
Imagem gerada no Nano Banna,
Animação da imagem feita no Animaker.
